# CASE01: The Long Form [Photograph] as an Algorithm

Concept | Programming Concept | Artistic Concept |
| --- | --- | --- |
| Function | The logic that connects photographs horizontally (`img`,`flex`, `overflow-x: scroll`) | the main arching narrative about 2 people meeting |
| Scope | `.env` -> `LOCATION=""` | The environment I'm in: framework (HTML/CSS/JS), fashion team, city/location, community-driven casting |
| Arguments | Individual photographs | The micro narrative, The casting, the styling  |
| Result | a final <img> | One continuous photographic work |
