# local-global-ppt

A VS Code-based presentation comprising my works and the discussion of locality & globality within them.

I ideated this alternative/interactive presentation mode to be more in the spirit of the event that this will be presented in -- Processing Community Day 2026. The `local/global` analogy reminds me a lot of programming syntax, and on top of that, Processing is a creative coding community, so it makes sense for me to not present in regular slides. Also it is just a cool idea, I think.

The presentation is in the format of `README.md`. Each file in `/notes` represent a 'slide'. On presentation mode, the README will be rendered on left column and the code part will be on the bottom right and the visualization of the code will be on the top right.

Presentation notes are in `/notes`. 
The code that accompanies each note is in `/code`. Each should be named the same as the note that it accompanies.
Related assets are organized according to the notes they are in in `/assets`.
